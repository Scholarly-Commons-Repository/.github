# Scholarly Commons Repository

> Git for scholarly knowledge.

SCR is a source-first repository ecosystem for scholarly texts.

Instead of distributing static PDF files, SCR focuses on preserving and maintaining the source materials behind scholarly works:

- Markdown
- TeX / LaTeX
- Bibliographic data
- Structured metadata
- Translation sources

Generated artifacts such as PDF, EPUB and HTML are treated as build outputs.

---

## Goals

### Preserve

Preserve public-domain and freely licensed scholarly works in editable source formats.

### Modernize

Allow historical works to be continuously improved:

- typo fixes
- formatting improvements
- updated references
- accessibility enhancements

### Collaborate

Enable Git-style workflows:

- version control
- pull requests
- forks
- reviews

### Build

Generate reproducible outputs:

- PDF
- EPUB
- HTML

from a common source tree.
